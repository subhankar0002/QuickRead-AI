# QuickRead AI

QuickRead AI is an AI-powered document summarization web app that allows users to upload PDF or DOCX files and instantly receive clear, concise summaries.  
It uses LangChain & Hugging Face llm model for natural language summarization, served via a Flask backend and an interactive HTML/CSS/JavaScript frontend.

---

## Features

- Upload PDF or DOCX files
- Generate AI-based summaries using Hugging Face models
- Skeleton loading animation for a smooth UX
- Dark mode toggle
- Download or copy the generated summary
- Clean and responsive UI
- Flask REST API integration

## About the Model

This project uses the *`facebook/bart-large-cnn`* model from Hugging Face for text summarization.
- Architecture: BART (Bidirectional and Auto-Regressive Transformer)  
- Type: Sequence-to-sequence model (encoder-decoder)  
- Pretraining: Trained on a large corpus of English text using denoising autoencoding  
- Fine-tuned On: CNN/Daily Mail dataset for summarization tasks  
- Task: Abstractive text summarization (it generates new sentences, not just extracts)
